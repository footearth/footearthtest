5.24. Gzip-1.3.12

Gzip 软件包包含压缩和解压文件的程序。

预计编译时间: 少于 0.1 SBU
所需磁盘空间: 2.2 MB

5.24.1. 安装 Gzip

该版本的 Gzip 使用的函数“futimens”和我们提供的 Glibc 版本不兼容。所以我们要重命名该函数： 

for file in gzip.c lib/utimens.{c,h} ; do \
   cp -v $file{,.orig}
   sed 's/futimens/gl_&/' $file.orig > $file
done

为编译作准备： 

./configure --prefix=/tools

编译软件包： 

make

现在编译完成了，同前面一样，在本章为了一个临时工具运行测试程序不是强制的。如果想运行测试套件，使用下列命令： 

make check

安装该软件包： 

make install

关于这个软件包的详细资料位于第 6.41.2 节 “Gzip 的内容”。
