6.33. File-4.26

File 是用来判断文件类型的工具。

预计编译时间: 0.1 SBU
所需磁盘空间: 8.9 MB

6.33.1. 安装 File

用 -e（-exclude）参数修复 man 手册使得它能够反映出最新的变化： 

sed -i -e '197,+1d' \
       -e '189,+1d' \
       -e 's/token$/tokens/' doc/file.man

为编译 File 做准备： 

./configure --prefix=/usr

编译软件包： 

make

要测试结果，请运行： 

make check

安装软件包： 

make install

6.33.2. File 的内容

安装的程序: file
安装的库: libmagic.{a,so}

简要描述

file

测试每一个指定的文件并且试图对它们进行分类。有三个测试集，按照下面的顺序执行：文件系统测试、幻数（magic number）测试、语言测试。第一个测试成功后会打印文件文件类型

libmagic

包含产生幻数（magic number）的函数，供 file程序使用
