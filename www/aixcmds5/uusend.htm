uusend 命令

用途

将文件发送到远程主机。

语法

uusend [ -mMode ] [ -r ] Sourcefile System [ !System ... ] ! RemoteFile

描述

uusend 命令把文件发送到远程系统上的给定位置。远程系统不必直接连接到本地系统上，但是 UUCP 链路的一个链必须连接这两个系统，并且 uusend 命令在链的每一个系统上都必须是可用的。

系统的链由 System[!System ...] 参数给定，它分别列出了文件要传送到的每一个远程系统，用 !（感叹号）隔开。!Remotefile 参数给出了要存储文件的名称（当它到达链中的最后一个系统时）。

注：不要在系统名称和感叹号之间或在最后一个感叹号和远程文件名称之间填加任何空格。

SourceFile 参数指定了在本地系统上文件的名称。如果使用了 -（短划线），则 uusend 命令使用标准输入。

标志

-m Mode 指定远程系统上文件的方式将从给定的八进制数字取得的。如果没有指定此标志，将会使用输入文件的方式。
-r 阻止启动会在系统间传输文件的 uucico 守护进程。缺省值是启动 uucico 守护进程。

当 uusend 命令在链中向下一个远程系统传送文件时，标志主要由该命令在内部使用。

示例

要将文件跨越一个系统传输到另一个系统，输入：

uusend /etc/motd nostromo!gandalf!~nuucp

/etc/motd 文件发送到系统 nostromo，然后到达系统 gandalf，并放在 nuucp 的主目录中 /var/spool/uucppublic/nuucp，其中 nuucp 是一个 BNU 登录标识。

文件

/usr/bin/uusend  
  包含 uusend 命令。
