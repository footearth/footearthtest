uninstallms 命令

用途

uninstallms － 卸载 Cluster Systems Management（CSM）管理服务器软件。

语法

uninstallms [-h] [-u] [-f] [-v | -V]

描述

该 uninstallms 命令卸载 CSM 管理服务器软件。该命令执行以下任务：

  • 除去节点组定义。
  • 使用 rmnode 命令除去群集节点定义。
  • 如果指定了 -u 标志，则该命令通过运行 rmnode -u 命令在节点上执行清除。 有关详细信息，请参考 rmnode 命令。
  • 除去预定条件。
  • 不要卸载任何 CSM 软件包。
  • 除去 CSM 日志文件。

由于 /csminstall 目录可能包含不属于 CSM 的数据，所以该命令不会删除其中的任何目录。

标志

-f
强制卸载管理服务器而不进行提示。
-h
显示使用信息。
-u
在卸载管理服务器前对每个节点运行 rmnode -u。
-v | -V
指定返回详细消息。

参数

无

安全性

该命令需要对于群集管理服务器的 root 用户权限。

退出状态

0
命令运行成功。
1
发生错误。

环境变量

RPOWER_LIST
指定一个文件，该文件包含主机的标识（每行一个，组成节点列表）。

标准错误

使用了 dshbak 过滤器且生成了标准错误消息时，有关标准错误的错误消息将在所有标准输出消息之前显示。无论是否使用 -c 标志，都将出现这种情况。

示例

卸载 CSM 管理服务器软件，输入：

uninstallms -u

位置

/opt/csm/bin/uninstallms

相关信息
