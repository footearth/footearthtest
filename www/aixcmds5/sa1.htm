sa1 命令

用途

将二进制数据收集并保存在 /var/adm/sa/sadd 文件中。

语法

/usr/lib/sa/sa1 [ Interval Number ]

描述

sa1 命令是 sadc 命令的 shell 过程变体，并处理该命令的所有标志和参数。 sa1 命令将二进制数据收集并存储在 /var/adm/sa/sadd 文件中，其中 dd 表示每月的日期。 Interval 和 Number 参数指定应以 Interval 秒的间隔，写入记录 Number 次。如果不指定这些参数，将写一个记录。要使用本命令，必须具有写入 /var/adm/sa 目录的许可权。

sa1 命令设计为由 cron 命令自动启动。如果 sa1 命令不是每天从 cron 命令运行，则 sar 命令显示 /usr/lib/sa/sa1 数据文件不存在的消息。

示例

要创建 sar 活动的每日记录，将以下条目放入 adm crontab 文件中：

0 8-17 * * 1-5 /usr/lib/sa/sa1 1200 3 &

文件

/var/adm/sa 指定包含每日数据文件的目录。
/var/adm/sa/sadd 包含每日数据文件，其中 dd 参数是一个表示每月日期的数字。
/usr/lib/sa/sa1 包含 sa1 命令。
