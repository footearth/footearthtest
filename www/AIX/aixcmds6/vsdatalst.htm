vsdatalst 命令

用途

vsdatalst - 显示虚拟共享磁盘子系统的信息。

语法

vsdatalst {-g | -n | -v | -c}

描述

使用此命令来向标准输出显示几种信息中的一种。

可以使用系统管理接口工具（SMIT）来运行 vsdatalst 命令。要使用 SMIT，输入：

smit list_vsd

并选择有关想要查看的虚拟共享磁盘 SDR 信息种类的选项。

标志

在每次调用 vsdatalst 时只能指定以下标志中的一个：

-g
显示以下全局卷组数据：
  • global_group_name、
  • local_group_name、
  • primary_server_node、
  • secondary_server_node。（这仅对可恢复的虚拟共享磁盘子系统启用。）
  • eio_recovery
  • recovery
  • CVSD server_list
-n
显示以下节点数据：
  • node_number、
  • host_name、
  • adapter_name、
  • min_buddy_buffer_size、
  • max_buddy_buffer_size、
  • max_buddy_buffers。
-v
显示以下定义数据：
  • vsd_name、
  • logical_volume_name、
  • global_group_name、
  • minor_number。
-c
显示以下群集信息：
  • node_number
  • cluster_name

参数

无。

安全性

必须具有 root 用户权限才能运行此命令。

退出状态

0
表示命令成功完成。
非零
表示出错。

限制

必须从对等域中的联机节点上发出此命令。要使对等域联机，请使用 startrpdomain 命令。要使特定节点在现有对等域中联机，请使用 startrpnode 命令。有关创建和管理 RSCT 对等域的更多信息，请参考 RSCT: Administration Guide。

标准输出

当前的 RVSD 子系统运行级别。

示例

  1. 要显示全局卷组日期，请输入： 
    vsdatalst -g
    系统会显示一条类似于以下的消息： 
    VSD 全局卷组信息
                                                服务器节点号
全局卷组名                   本地 VG 名    主卷组   备份卷组 eio_recovery   恢复    server_list  vsd_type
------------------------    ------------   -------   ------ ------------  --------  -----------  --------
gpfs0gvg                     gpfs0vg         1         2          0          0        0            VSD
gpfs1gvg                     gpfs1vg         2         1          0          0        0            VSD
gpfs3gvg                     gpfs3vg         1         0          0          0        1:2          CVSD
  2. 要显示全局卷组日期，请输入： 
    vsdatalst -n
    系统会显示一条类似于以下的消息： 
    VSD 节点信息
                                               伙伴缓冲区
  节点                 VSD      IP 信息包  最小    最大   最大缓冲区数
编号 host_name       适配器     大小     大小    大小
------ --------------- -------- --------- ------- ------- ---------
     1 host1            ml0       61440    4096   262144    128
     2 host2            ml0       61440    4096   262144    128
  3. 要显示全局卷组日期，请输入： 
    vsdatalst -v
    系统会显示一条类似于以下的消息： 
    VSD 表
VSD 名称                        逻辑卷          全局卷组                       次要编号 size_in_MB
------------------------------- --------------- ------------------------------- ------ ----------
gpfs0vsd                        gpfs0lv         gpfs0gvg                             3 4096
gpfs1vsd                        gpfs1lv         gpfs1gvg                             1 4096
gpfs3vsd                        gpfs3lv         gpfs3gvg                             4 4096

位置

/opt/rsct/vsd/bin/vsdatalst
