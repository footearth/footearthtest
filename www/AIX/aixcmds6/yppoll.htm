yppoll 命令

用途

显示当前服务器上使用的网络信息服务（NIS）映射的顺序号（标识号）。

语法

/usr/sbin/yppoll [ -h Host ] [ -d Domain ] MapName

描述

yppoll 命令使用 ypserv 守护进程显示 以 MapName 参数指定的映射的顺序号。顺序号 是映射的标识号，该号码由系统分配。每当映射被更新时，这个号码就会更改。在想确定服务器使用的是特定映射的最新版本时，请使用 yppoll 命令。

yppoll 命令可以在安装了网络信息服务（NIS）协议的版本 1 或版本 2 的系统上运行。不过 要注意的是，协议的每一版本都有它自己的诊断消息设置。

注:
指定 MapName 时，必须确保输入映射的全名。yppoll 命令不能识别映射的别名。

标志

-h Host 使您可以指定一个服务器而不是使用缺省值服务器。要找出命令缺省使用的服务器，请使用 ypwhich 命令。
-d Domain 使您可以指定一个域而不是缺省域。要找出命令缺省使用的域，请使用 domainname 命令。

示例

  1. 要察看在特定主机上的映射，输入： 

    /usr/sbin/yppoll -h thor netgroups.byuser

    在本示例中，yppoll 命令显示位于名为 thor 的主机上的netgroups.byuser 映射的顺序号。

  2. 要察看域上的映射，请输入： 

    /usr/sbin/yppoll -d atlantis hosts.byname

    在该示例中，ypoll 命令显示位于 atlantis 域 的 hosts.byname 映射的顺序号。
