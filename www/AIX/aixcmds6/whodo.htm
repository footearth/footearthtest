whodo 命令

用途

列出系统中用户正执行的任务。

语法

whodo [ -h ] [ -l ] [ User ]

描述

打印关于一个终端的所有进程及子进程的信息。

缺省情况下，每个活动登录用户的命令所生成的 输出将会包含终端名、用户标识、登录日期和时间。输出以日期、时间和机器 名开头。这个信息之后是一个与那个用户标识相关的活动进程记录。每个 记录显示终端名、进程标识、使用的 CPU 分秒数和进程名。

标志

-h 禁止打印到输出上的标题。
-l 生成长格式的输出。打印系统上当前活动的 一个摘要。摘要包含以下内容：
用户
登录用户。
tty
用户使用的 tty 名称。
登录 @
用户登录日期和时间。
空闲
自程序上次试图读取终端开始持续的分钟数。
JCPU
该终端上的所有进程及其子进程使用的系统部件时间。
PCPU
当前活动进程所使用的系统部件时间。
什么
当前进程的名称和参数。
摘要标题行显示当前日期和时间、 系统已经启动的时间、登录到系统的用户数。

参数

User 限制为仅输出与 User 指定的用户 相关所有会话。一次指定的用户名不能超过一个。

退出状态

0
命令成功完成。
>0
发生一个错误。
