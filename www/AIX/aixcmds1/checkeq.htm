checkeq 或 checkmm 命令

用途

检查使用备忘录宏进行格式化的文档。

语法

{ checkeq | checkmm } [ File... ]

描述

checkeq 命令用以检查已为 neqn 或 eqn 命令准备的指定文件（File）中的语法错误。checkeq 命令报告丢失或不平衡定界符以及 .EQ 和 .EN 宏对。

在功能上，checkeq 命令等同于 checkmm 命令。

checkmm（检查备忘录宏）命令用以检查已为 mm 命令或 mmt 命令准备的文件中的语法错误。例如，checkmm 命令检查是否具有与每个 .DS（显示开始）宏相对应的 .DE（显示结束）宏。File 指定将由 checkeq 或 checkmm 命令检查的文件。

checkmm 命令的输出是所检查行的行数以及由于丢失的宏而未完成的宏的列表。
