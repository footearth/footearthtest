configassist 命令

用途

显示配置助手向导。

语法

/usr/websm/bin/configassist

描述

配置助手向导在安装操作系统后自动显示，并用来帮助完成配置任务。它也能在任何时间运行来完成额外的配置。使用配置助手来配置一个安装了 HTTP 服务器以在浏览器上运行基于 Web 的系统管理器的系统。参见第 4 页的 Applet Mode 以获得更多信息。

注:
本命令的全路径名 /usr/websm/bin/configassist 必须指定。

标志

无

示例

不可用

相关信息

要获得安装基于 Web 的系统管理器的信息，参见第2章：“安装和系统要求” ，在 《AIX 5L V5.2 基于 Web 的系统管理器管理指南》 中。
