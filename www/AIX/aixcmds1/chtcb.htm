chtcb 命令

用途

更改或查询文件的可信计算基属性。

语法

chtcb { on | off | query } File ...

描述

chtcb 命令更改或查询您使用 File 参数指定的文件的可信计算基（TCB）属性。以下选项有效：

on 启用可信计算基属性。
off 禁用可信计算基属性（如果已设置）。
query 显示可信计算基属性的值。

此命令应该在可信路径上执行。

安全性

访问控制：命令应授权执行（x）访问权限给 root 用户和安全组的成员。该命令应具有可信计算基属性。

示例

  1. 要将 plans 文件识别为可信计算基（TCB）的一部分，则通过输入以下命令将可信计算基属性设置为 on 值： 

    chtcb on plans

    plans 文件现在可以从可信路径执行。

  2. 要查询 plans 文件是否为可信计算基（TCB）的一部分，请输入： 

    chtcb query plans

    当出现状态时，如果可信计算基属性设置为 on 值，则知道 plans 文件是 TCB 的一部分。

  3. 要从可信计算基（TCB）除去 plans 文件，请输入： 

    chtcb off plans

文件

/usr/sbin/chtcb 包含 chtcb 命令。
