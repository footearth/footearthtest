bterm 命令

用途

以双向（BIDI）方式仿真终端。

语法

bterm [ -maps Map ] [ -help ] [ -keywords ] [ -nobidi ] [ -symmetric ] [ -autopush ] [ -or Orientation ] [ -text TextType ] [ -nss NumShape ] [ -csd CharShape ] [ -tail ] [ -nonulls ]

描述

bterm 命令仿真 IBM 3151、VT220、HFT 和其它终端。它是以 BIDI 方式在 ASCII 终端上运行。此命令会创建能运行任何 BIDI 应用程序的 BIDI shell。不能从其内部递归地启动 bterm 命令。

确定键盘映射和字符对称交换的映射由 -maps 标志指定。可以使用 bterm 命令可用的标志或者通过在缺省文件中设置 BIDI 行为来指定其它 BIDI 行为。这样的行为包括缺省文本方式、缺省屏幕定向、缺省阿拉伯字符整形、缺省数字整形、“对称交换”方式是否已启用、自动推（Autopush）方式是否已启用。用标志指定的行为先于在缺省文件中设置的行为。

以下列顺序搜索缺省文件：

  1. 在主目录中搜索 .Bidi-defaults 文件。
  2. 如果未找到该文件，bterm 命令会搜索 /usr/lib/nls/bidi/$LANG/app-defaults 文件中的 BTerm 资源文件。

标志

-autopush 以可见文本方式启用自动推（Autopush）方式。
-csd CharShape 指定阿拉伯字符的整形。CharShape 变量可以是以下选项之一：
  • automatic
  • isolated（仅可见文本方式）
  • initial（仅可见文本方式）
  • middle（仅可见文本方式）
  • final（仅可见文本方式）
  • passthru

    缺省值为 automatic 整形。
-help 列出可用的参数及其语法。
-keywords 列出缺省文件中可用的关键字。
-maps Map 指定用于键盘映射和字符对称交换的映射。每种语言有不同映射，且 Map 变量可用的选项位于 /usr/lib/nls/bidi/maps 目录中。必须如下所示指定环境变量 BIDIPATH： 

export BIDIPATH=/usr/lib/nls/bidi
-nobidi 禁用 BIDI 方式。
-nonulls 使用空格而非空来初始化屏幕。
-nss NumShape 指定数字的整形。为 NumShape 变量指定以下选项之一：
  • bilingual
  • hindi
  • arabic
  • passthru

缺省值为 bilingual。
-or Orientation 指定屏幕定向。Orientation 变量可以是 LTR 或 RTL。缺省值为 LTR。
-symmetric 启用“对称交换”方式。
-tail 以两个单元而非一个单元的格式写阿拉伯语的“seen”、“sheen”、“sad”和“dad”字符。
-text TextType 指定数据流类型。TextType 变量可以是 implicit 或 visual。缺省值为 implicit。

键组合

要使用键组合更改 BIDI 设置，按 Ctrl+X 按键顺序以输入 BIDI 命令方式。在此按键顺序后输入的任何键都被解释为 BIDI 命令。无效键会引起蜂鸣声并退出 BIDI 命令方式。以下键是有效的 BIDI 命令：

键 用途
r 逆向屏幕定向。
n 将语言键盘层设置为本地语言。
l 将语言键盘层设置为拉丁语。
a 切换阿拉伯字符的自动整形变量选项（也对隐式方式有效）。
t 显示状态。
space 输入所要求的空格（RSP）。

仅对隐式方式：

键 用途
c 切换列标题方式。

仅对可见方式：

键 用途
s 启动推（Push）方式。
e 终止结束推（End Push）方式。
p 切换自动推（Autopush）方式。
f 以阿拉伯字符的最后形式对这些字符进行整形。
i 以阿拉伯字符的初始形式对这些字符进行整形。
b 以 Passthru 方式对阿拉伯字符进行整形。
o 以阿拉伯字符的分离形式对这些字符进行整形。
m 以阿拉伯字符的中间形式对这些字符进行整形。

off 禁用自动推（AutoPush）功能键。
fASD on 启用“自动整形确定”功能键。
off 禁用“自动整形确定”功能键。
fShapeIS on 启用“分离整形”功能键。

numShape bilingual 将数字整形设置为双语。

middle 以中间方式显示阿拉伯字符。
maps

expandTail on 将类似“seen”的字符及它们的尾部写在两个单元上。
off 将类似“seen”的字符及它们的尾部写在一个单元上。
nobidi on 激活 BIDI 方式。
off 关闭 BIDI 方式。
noNulls on 用空格替代空。
off 将空保留为空，不用空格替代。
