chdoclang 命令

用途

为系统或用户设置缺省文档语言。

语法

chdoclang [ -d ] [ -u UID | Uname ] Language

描述

chdoclang 命令通过添加环境变量定义至 /etc/environment 或用户的 .profile 文件来为系统或用户设置缺省文档语言。当用户打开文档库时，将使用用户的缺省文档语言而不是系统广义设置。

如果使用 docsearch 命令或 CDE 桌面中的 Documentation Library 图标启动文档库应用程序，此应用程序就是以这种语言出现的。

标志

-d 除去先前的缺省文档语言设置。
-u 为指定用户进行修改。

示例

  1. 要将缺省文档语言更改为英语，输入： 

    chdoclang en_US
  2. 要为用户 fred 将缺省文档语言更改为日语，输入： 

    chdoclang -u fred Ja_JP
  3. 要为用户标识为 201 的用户将缺省文档语言更改为德语，输入： 

    chdoclang -u 201 de_DE

文件

/usr/bin/chdoclang 更改文档语言命令
/etc/environment 指定所有进程的基本环境
$HOME/.profile 指定特定的用户需要的环境
