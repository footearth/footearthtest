chnfs 命令

用途

更改系统的配置以调用 biod 和 nfsd 守护进程的指定数。

语法

/usr/sbin/chnfs [ -n NumberOfNfsd ] [ -b NumberOfBiod ] [ -I | -B | -N ]

描述

chnfs 命令调用指定的 biod 和 nfsd 守护进程数。chnfs 命令通过更改 SRC 数据库中的对象来进行此操作。取决于选择的标志，这些更改在不同的时间发生。

标志

-B 临时停止当前在系统上运行的守护进程，修改 SRC 数据库代码以反映新数字，并重新启动指示的守护进程。这个标志是缺省的。
-b NumberOfBiod 指定要在系统上运行的 biod 守护进程的数目。
-I 更改 SRC 数据库中的对象，以便指定的守护进程数将在下一次系统重新启动期间运行。
-N 临时停止当前在系统上运行的守护进程并重新启动指示的守护进程数。
-n NumberOfNfsd 指定要在系统上运行的 nfsd 守护进程的数目。

示例

要将 nfsd 守护进程数设置为 10，biod 守护进程数设置为 4，请输入：

chnfs    -n    10    -b    4    -I

将为下一次系统重新启动进行此更改。
