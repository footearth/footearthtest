nddctl 命令

用途

把命令发布到网络设备驱动程序（NND）。

参数

nddctl { -r } Device

描述

The nddctl 命令允许用户在运行时控制 NND 设备（即无需重新配置设备驱动程序，而这个过程通常需要中断网络连接）。

标志

-r 在运行时，强制 NDD 设备重新协商它的链接属性（速度和双重性）。
注:
强制连接重新协商需要复位该设备；在该设备重新初始化时，这可能会造成持续数秒钟的网络连接损失。

参数

设备 指定执行指定的命令的 NDD 设备。

退出状态

0 命令成功完成。
>0 有错误发生。

示例

  1. 要在运行时强制设备 ent0 重新协商它的连接属性，请输入： 
    nddctl -r ent0

位置

/usr/lib/methods
