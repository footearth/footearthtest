rmvirprt 命令

用途

除去虚拟打印机。

语法

rmvirprt -q PrinterQueueName -d QueueDeviceName

描述

rmvirprt 命令除去分配给 PrinterQueueName 和 QueueDeviceName 变量值的虚拟打印机。rmvirprt 命令也除去与指定队列和队列设备有关的系统管理界面程序（SMIT）对象数据库管理器（ODM）。

可以使用 基于 Web 的系统管理器 中的 打印机队列 应用程序更改打印机特征。

也可以使用系统管理界面程序（SMIT） smit rmvirprt 快速路径运行此命令。

注:
当从命令行运行 rmvirprt 命令， 不但不除去队列或队列设备，而且也不检查任何 正在运行或已经在指定队列和队列设备上排列等待的作业。 但是，如果使用 SMIT 交互的运行该命令，如果没有正在运行或排队等待的作业，则相应的队列、队列设备和打印机设备（可选择的） 与虚拟打印机一起除去。

标志

-d QueueDeviceName 指定分配有虚拟打印机的队列设备的名称。
-q PrinterQueueName 指定分配有虚拟打印机的打印队列的名称。

示例

要除去与 proq 打印机队列相关的 mypro 虚拟打印机的属性值，输入：

rmvirprt  -d mypro  -q proq

文件

/etc/qconfig 包含配置文件。
/usr/sbin/rmvirprt 包含 rmvirprt 命令。
/var/spool/lpd/pio/@local/custom/* 包含定制的虚拟打印机属性文件。
/var/spool/lpd/pio/@local/ddi/* 包含虚拟打印机摘要文件。
