prtconf 命令

用途

显示系统配置信息。

语法

prtconf [ -c ] [ -k ] [ -L ] [ -m ] [ -s ] [ -v ]

描述

如果不带任何标志运行 prtconf 命令， 会显示系统型号、机器序列号、处理器类型、处理器数目、处理器时钟速度、cpu 类型、总内存大小、网络信息、文件系统信息、调页空间信息和设备信息。

标志

-c 显示 cpu 类型，例如 32－bit 或 64－bit。
-k 显示正在使用的内核，例如，32－bit 或 64－bit。
-L 如果它是 LPAR 分区，显示 LPAR 分区数和分区名，否则返回“-1 NULL”。
-m 显示系统内存。
-s 以兆赫兹为单位显示处理器时钟速度。
-v 显示设备的定制 VPD 对象类中找到的 VPD。

退出状态

0
命令成功完成。
>0
发生错误。

    prtconf -k
    系统显示如下的内核类型信息：

    prtconf -m
