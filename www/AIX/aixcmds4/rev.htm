rev 命令

用途

逆向文件中的每一行字符。

语法

rev [ File ... ] 

描述

rev 命令复制指定文件到标准输出，逆向在每一行里的字符顺序。如果没有指定文件，rev 命令读取标准输入。

示例

要逆向在文件中的每一行字符，输入：

rev file

如果 file 文件包含文本：

abcdefghi
123456789

那么 rev 命令显示：

ihgfedcba
987654321

文件

/usr/bin/rev 包含 rev 命令。
