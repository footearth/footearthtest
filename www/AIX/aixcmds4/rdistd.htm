rdistd 命令

用途

用于远程发布文件的服务器程序。

语法

rdistd -S

rdistd -V

描述

rdistd 是 rdist 命令的服务器程序。它以 rdist 通过 rsh 正常运行。

-S 标志确保 rdistd 不会意外启动，因为它通常驻留在用户正常的 PATH 环境变量中。

标志

-V 打印版本信息并退出。

退出状态

这个命令返回下列出口值：

0
成功完成。
>0
发生错误。

文件

/usr/sbin/rdistd 包含 rdistd 服务器
/usr/bin/rdistd 到 /usr/sbin/rdistd 的符号链路

相关信息
