rmtun 命令

用途

取消激活操作的隧道和有选择的除去隧道定义。

语法

rmtun -v 4|6 -t tid_list | all [-d]

描述

使用 rmtun 命令取消激活活动的隧道和有选择的除去隧道定义。 在从隧道数据库中除去隧道定义时，也将由 gentun 命令除去为隧道创建的自动生成的过滤器规则。

标志

all 取消激活和有选择的除去所有的隧道。
tid_list 想取消激活的隧道列表。隧道标识由“，”或“－”分开。 可以使用“－”指定标识的范围。例如，指定 1,3,5-7，在列表中有五个隧道标识， 1, 3, 5, 6 和 7。
-d 指定隧道将从隧道数据库除去。这是可选的标志。
-t 要取消激活的隧道列表。如果指定 -d，列表中的所有隧道定义也将从隧道数据库除去。
-v 隧道的 IP 版本。对于 IP V4 隧道，使用值 4。 对于 IP V6 隧道，使用值 6。
