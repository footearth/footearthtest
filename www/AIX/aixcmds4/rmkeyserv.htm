rmkeyserv 命令

用途

停止 keyserv 守护程序并注释 /etc/rc.nfs 文件中的条目。

语法

/usr/sbin/rmkeyserv [  -I | -B | -N ]

描述

rmkeyserv 命令注释 /etc/rc.nfs 文件中 keyserv 守护程序的条目。 通过使用 stopsrc 命令，rmkeyserv 守护程序停止 keyserv 守护程序。

可以在 基于 Web 的系统管理器（wsm）中使用 文件系统 应用程序来更改文件系统的特征。

也可以使用系统管理界面程序（SMIT）smit rmkeyserv 快速路径运行该命令。

标志

-I 注释 /etc/rc.nfs 文件中 keyserv 守护程序的条目。
-B 注释 /etc/rc.nfs 文件中 keyserv 守护程序的条目并停止 keyserv 守护程序。 这个标志是缺省的。
-N 使用 stopsrc 命令停止 keyserv 守护程序。 该标志不更改 /etc/rc.nfs 文件。

示例

要注释 /etc/rc.nfs 文件中启动 keyserv 守护程序的条目，请输入：

rmkeyserv -I

该命令将不停止当前正在执行的守护程序。

文件

/etc/rc.nfs 包含 NFS 和 NIS 守护程序的启动脚本。
