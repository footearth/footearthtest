readlvcopy 命令

用途

读取一个特定的逻辑卷的镜像副本。

语法

readlvcopy -d device [ -c copy | -C copy | -b ] [ -n number_of_blocks ] [ -o outfile ] [ -s skip] [ -S seek ]

描述

标志

-d device 从逻辑卷特殊设备文件读取
-c copy 要从中读取所请求的镜像副本。对第 1、2 和第 3 个数据副本的有效值是 1、2 和 3。即使逻辑分区被标记为旧文件也要读取数据。缺省值是数据的第一份副本。
-C copy 要从中读取所请求的镜像副本。对第 1、2 和第 3 个数据副本的有效值是 1、2 和 3。不读取旧文件逻辑分区。
-b 读取被标记为联机备份的镜像副本。
-n number_of_blocks 要读取的大小为 128K 的块的数目
-o outfile 目标文件。缺省是 stdout
-s skip 要跳进 device 的大小为 128K 的块的数目。
-S seek 要查寻进入 outfile 的大小为 128K 的块的数目。
