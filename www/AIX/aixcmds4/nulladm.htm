nulladm 命令

用途

创建活动记帐数据文件。

语法

/usr/sbin/acct/nulladm [ File ... ]

描述

此 nulladm 命令创建由 File 参数指定的文件，并给文件所有者和组读（r）和写（w）的权限，给其它用户读的权限，以确保文件所有者和组是 adm。不同的记帐 shell 过程调用 nulladm 命令。具备管理权限的用户能够使用此命令设置活动数据文件，如 /var/adm/wtmp 文件。

注意： 在分布式环境中，不要在节点间共享记帐文件。 每个节点应该有自己的不同的记帐文件的副本。

安全性

访问控制：此命令只授权给 adm 组成员执行访问。

文件

/usr/sbin/acct 包含记帐命令。
/var/adm/acct/sum 包含记帐数据文件。
