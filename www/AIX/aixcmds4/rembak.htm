rembak 命令

用途

发送打印作业至远程服务器上的队列。

语法

rembak -S Server -P Queue [ -R ] [ -N Filter] [ -L ] [ -q ] [ -x ] [ -# JobNumber ] [ -u UserName ] [ -X ] [ -o Option ] [ -T Timeout] [ -C] [ -D DebugOutputFile] [ File ... ]

描述

rembak 命令发送一个作业至远程服务器上的队列。请求可以是打印作业、状态请求、作业取消请求或杀死远程队列系统的请求中的任何一个。服务器和队列标志是必需的。全部其它标志是可选的，取决于要完成什么操作。

该命令应该只被 qdaemon 命令调用。其意图不是让用户在命令行上输入此命令。关于如何发出打印作业请求或如何使用系统管理器接口工具（ SMIT）请求打印作业的详细信息，请参阅 enq 命令。

标志

-# JobNumber 指定取消 JobNumber。
-C 首先发送控制文件。lpd 协议允许两个握手序列处理一个打印作业。缺省值是先发送数据文件，然后是控制文件。其它序列是先控制文件，然后是数据文件。如果指定 -C，rembak 将首先发送控制文件，然后是数据文件。
-D DebugOutputfile 为 rembak 打开调试选项。如果不指定输出文件名称，或者创建或写到输出文件时遇到任何问题，调试选项会被忽略。如果指定的输出文件已经存在，会在其尾部附加新的调试输出。
-L 指示来自远程队列的长（详细）状态请求。
-N Filter 指示远程服务器的机器类型。过滤器名称由 s_statfilter 属性指定，该属性在 /etc/qconfig 文件中。filter 变量的值包含下列：
/usr/lib/lpd/aixshort
指示服务器是另一个 AIX 机器。
/usr/lib/lpd/aixv2short
指示服务器是 RT，具有 AIX 版本 2 操作系统。
/usr/lib/lpd/bsdshort
指示服务器是 bsd 机器。
/usr/lib/lpd/attshort
指示服务器是 AT&T 机器。
-o Option 指定 Option 选项发送到远程服务器后端。（这些 Options 通过 rembak 命令传递）。
-P Queue 指定远程服务器上的 Queue 的名称，打印作业将发送到该远程服务器。
-q 表示来自远程队列的简短（简略）状态请求。
-R 重新启动远程队列系统。

注：发送请求到操作系统时，不支持 -R 标志。lpd 守护程序并不支持这样的请求。 支持 -R 标志仅是为了和其它系统兼容。
-S Server 指定远程打印Server 的名称，打印作业将发送到该远程服务器。
-T Timeout 为 rembak 等候来自远程服务器的确认设置超时周期（以分钟为单位）。如果不指定值，则使用 90 秒缺省超时。如果超时值是零或是负值，也会使用该缺省值。
-u UserName@HostName 为 UserName 取消从 HostName 机器提交的打印作业。

注：队列系统不支持多字节主机名。
-X 指定 rembak 命令发送 -o Option 到远程服务器，即使该远程服务器不是 AIX 机器。如果远程服务器不是 AIX 机器，那么不带 -o 标志发送 Option。这样，-o -abc 将作为 -abc 发送。

要在远程队列上使用 -X 标志，必须为特定队列在 /etc/qconfig 文件中包含以下行：

backend = /usr/lib/lpd/rembak -X

如果在队列上指定 -X，不能保证 qprt、lpr 和其它队列命令起作用。使用 enq 命令。
-x 取消作业请求。使用 -# JobNumber 标志或 -u UserName 标志取消请求。

示例

  1. 要打印远程服务器 olive 上的队列 popeye 中的 spinach、asparagus 和 broccoli 文件，该远程服务器是具有 AIX 版本 2 操作系统的 RT，请输入： 

    rembak -S olive -P popeye -N /usr/lib/lpd/aixv2short spinach asparagus broccoli
  2. 要发出一个详细状态请求至 olive 上的 popeye 队列，请输入： 

    rembak -S olive -P popeye -N /usr/lib/lpd/aixv2short -L
  3. 要在远程服务器上取消用户 sweetpea 从机器 bluto 机器提交的作业号 23， 该机器是一个版本 3 机器，请输入： 

    rembak -S olive -P popeye -N /usr/lib/lpd/aixv2short -x -#23 -u sweetpea@bluto

文件

/usr/lib/lpd/rembak 包含 rembak 命令。
/etc/hosts.lpd 包含允许执行打印请求的主机名。
/etc/hosts.equiv 包含允许执行打印请求的主机名。
