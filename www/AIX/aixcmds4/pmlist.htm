pmlist 命令

用途

列出有关受支持的处理器的信息。

语法

pmlist [ -h ]

pmlist [ -o t | c ] -l

pmlist [ -o t | c ] [ -p ProcessorType ] [ -s ] [ -d] [ -f Filter ] [ -e ShortName | -c Counter[,Event ] | -g Group ]

描述

pmlist 命令执行以下功能：

  • 列出受支持的处理器。
  • 列出指定处理器的信息摘要。
  • 列出指定处理器的事件表。
  • 列出指定处理器的任何现有事件组。

标志

-c -1 列出全部计数器的所有事件。
-c Counter 列出指定 Counter 的所有事件。
-c Counter,Event 列出指定 Counter 的指定 Event。
-d 显示事件的详细描述。
-e ShortEvent 列出所有“计数器”的指定 ShortName 的描述。
-f v,u,c 将事件过滤器指定为逗号分隔的过滤器列表。有效过滤器为：v（已验证）、u（未验证）和 c（警告）。 这些过滤器提供事件的测试状态。缺省过滤器为 v,u,c。
-g -1 列出所有事件组。
-g Group 列出指定的事件 Group。
-h 显示 pmlist 命令的帮助信息。
-l 列出所有受支持的处理器类型。
-o t|c 指定 pmlist 命令的输出格式。有效的输出格式指定为以下格式之一：t（文本格式）、h（HTML 格式）和 c（CSV 格式）。缺省的输出格式为文本格式。
-p ProcessorType 指定处理器类型。
-s 显示处理器信息摘要。

示例

  1. 要显示所有受支持构成的处理器列表，请输入： 
    pmlist -l
  2. 要显示当前处理器的摘要信息，请输入： 
    pmlist -s
  3. 要以 CSV 格式显示当前处理器的摘要信息，请输入： 
    pmlist -s -o c
  4. 要显示组号为 62 的当前处理器（如果当前处理器支持事件组），请输入： 
    pmlist  -g 62
  5. 要显示 POWER4 处理器的计数器 1 的事件 3 的详细信息，请输入： 
    pmlist -p POWER4 -c 1,3 -d
