prdaily 命令

用途

创建一个前一天记帐数据的 ASCII 报告。

语法

/usr/sbin/acct/prdaily [  -l ] [ mmdd ] [  -c ]

描述

prdaily 命令由 runacct 命令调用生成前一天记帐数据的 ASCII 报表。这个报告驻留在 /var/adm/acct/sum/rprtmmdd 文件中， 这里 mmdd 指定报告的月份日期。

标志

-c 根据命令报告异常资源使用情况。这个标志只能被当天的记帐数据使用。
-l [mmdd] 根据登录标识报告指定日期的异常使用情况。使用 mmdd 变量可指定当天以外的日期。

安全性

访问控制：这个命令仅将执行（x）访问权限授权给 adm 组成员。

文件

/usr/sbin/acct 记帐命令的路径。
/usr/sbin/acct/ptelus.awk 计算登录标识的异常使用的限定。这是一个 shell 步骤。
/usr/sbin/acct/ptecms.awk 计算命令名称的异常使用的限定。这是一个 shell 步骤。
/var/adm/acct/sum 为每日记帐记录添加目录。

相关信息
