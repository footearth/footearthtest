rmyp 命令

用途

除去 NIS 的配置。

语法

/usr/sbin/rmyp {  -s | -c }

描述

rmyp 命令从系统除去用来使 NIS 工作的所有东西。例如，rmyp 命令除去所有的 NIS 映射和 NIS 守护程序的 /etc/rc.nfs 文件的所有项。

可以使用 基于 Web 的系统管理器（wsm）的 网络 应用程序更改网络特征。

也可以使用系统管理界面程序（SMIT）smit rmyp 快速路径运行该命令。可以使用 系统管理界面程序（SMIT） 运行该命令。要使用 SMIT，输入：

smit rmyp

标志

-s 从系统除去服务器配置。
-c 从系统除去客户机配置。
