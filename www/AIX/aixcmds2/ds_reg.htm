ds_reg 命令

用途

文档库服务注册工具

语法

ds_reg [ -q ] -l Locale ViewSet View
ds_reg [ -q ] [ -d ] Locale ViewSet View ViewDefinitionFile

描述

在缺省情况下，对于指定的语言环境，ds_reg 命令采用给定视图集中指定的视图，注册了一个视图定义文件的内容。
通过 -d 标志，ds_reg 命令从为指定语言环境给定的视图集中的指定视图，除去了一个视图定义文件的以前注册内容。
通过 -l 标志，ds_reg 命令在对于指定语言环境所给定的视图集中列出指定的视图。

标志

-d 从为指定语言环境所给定的视图集中，取消对视图定义文件内容的注册。
-l 列出给定视图集和语言环境中指定视图的内容。不需要指定视图定义文件。
-q 指定 quiet 选项，使得 ds_reg 禁止消息的显示。

示例

要将视图定义文件‘MyBook.vdf’的内容注册到英语（en_US）语言环境中‘Global’视图集的‘Books’视图，请输入：

ds_reg en_US Global Books MyBook.vdf

要从英语（en_US）语言环境中‘Global’视图集的‘Books’视图除去视图定义文件‘MyBook.vdf’的内容，请输入：

ds_reg -d en_US Global Books MyBook.vdf

要列出英语（en_US）语言环境中‘Global’视图集的‘Books’视图的内容，请输入：

ds_reg -l en_US Global Books

文件

/usr/sbin/ds_reg ds_reg 命令

/usr/lib/nls/msg/$LANG/docsearch.cat 文档库消息编目
