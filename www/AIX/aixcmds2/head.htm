head 命令

用途

显示一个文件或多个文件的前几行或前几个字节。

语法

head [  - Count  |  -c Count | -n  Number  ] [  File  ... ]

描述

head 命令把每一指定文件或标准输入的指定数量的行或字节写入标准输出。如果不为 head 命令指定任何标志, 缺省显示前 10 行。File参数指定了输入文件名。输入文件必须是文本文件。当指定多个文件时每一文件的开始应与下列一致。

==> 文件名 <==

显示一组短文件并对每一个进行识别，请输入：

example% head -9999 filename1 filename2...

标志

-Count 从每一要显示的指定文件的开头指定行数。Count变量必须是一个正的十进制整数。此标志等价于 -n Number标志，但如果考虑到便携性，就不应该使用。
-c End 指定要显示的字节数。Number变量必须是一个正的十进制整数。
-n Number 指定从每一要显示的指定文件的开头的行数。Number变量必须是一个正的十进制整数。此标志等价于 - Count 标志。

退出状态

此命令返回下列出口值:

0 成功完成。
>0 发生错误。

示例

显示Test文件的前 5 行，请输入：

head -5 Test

或者

head -n 5 Test
