mkquedev 命令

用途

把打印机队列设备添加到系统中。

语法

mkquedev -d Name -q Name -a 'Attribute = Value' ...

描述

通过把命令行中描述的节添加到 /etc/qconfig 文件，mkquedev 命令把打印机队列设备添加到系统中。

可以在“基于 Web 的系统管理器”（wsm）中使用打印机队列应用程序来改变打印机特征。也可以使用“系统管理接口工具”（SMIT）smit mkquedev 快速路径来运行该命令。

注：在任何队列中有活动的作业时请不要编辑 /etc/qconfig 文件。编辑包括手工编辑和使用 chque、mkque、rmque、mkquedev、rmquedev 或 chquedev 命令。建议使用这些命令对 /etc/qconfig 文件进行所有的更改。不过，如果想要手工编辑，则首先要发出 enq -G 命令，在处理了所有的作业后，使排队系统和 qdaemon 停机。然后编辑 /etc/qconfig 文件并以新配置重新启动 qdaemon。

标志

-a 'Attribute = Value' 指定 'Attribute = Value' 属性以添加到 /etc/qconfig 文件的设备节中。当在命令行输入 mkquedev 命令时，该标志必须是最后的标志。有关有效属性的列表，请参阅 /etc/qconfig 文件。

注：当在命令行输入该命令时，必须包含 'backend = ' 属性。
-d Name 用 Name 变量指定要添加的队列设备的名称。

注：队列设备名一定不能超过 20 个字符。
-q Name 用 Name 变量指定队列设备添加到的队列名称（该名称必须已经存在）。mkquedev 命令自动把 'device = ' 属性添加到指定的队列节。

示例

要把 postscript 打印队列 设备添加到 lp0 队列，指定后端程序为 piobe 命令（backend = /usr/lib/lpd/piobe）并且指示后端程序不要把纸对齐（align = false），请输入：

mkquedev  -qlp0  -dpostscript  -a 'backend = /usr/lib/lpd/piobe' \
-a 'align = FALSE'

文件

/usr/bin/mkquedev 包含 mkquedev 命令。
/etc/qconfig 包含配置文件。
