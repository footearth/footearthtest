ipsecstat 命令

用途

列出 IP 安全设备的状态、IP 安全加密算法和 IP 安全信息包的统计信息。

语法

ipsecstat [ -c ] [ -d ] [ -A ] [ -E ]

描述

ipsecstat 命令（不带标志使用）显示 IP 安全设备的状态、为 IP 安全安装的加密算法和 IP 安全信息包的统计信息。

这个命令可以带上标志使用，以仅列出 IP 安全设备的状态、已安装的算法或者重新设置统计信息计数器（为 0）。

标志

-c 重新设置统计信息计数器（在显示当前值后）。-c 标志不能和任何其它标志一起使用。
-d 仅列出 IP 安全设备的状态。-d 标志不能和其他任何标志一起使用。
-A 仅列出已安装的认证算法。 -A 标志不能和其它任何标志一起使用。
-E 仅列出已安装的加密算法。-E 标志不能和任何其它标志一起使用。
