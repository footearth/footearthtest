lsrset 命令

用途

显示系统 rset 内容。

语法

lsrset [ -f ] [ -v | -o ] [ -r rsetname | -n namespace | -a ]

或

lsrset [ -P ] [ -v | -o ] -p pid

描述

lsrset 命令显示存储在系统注册表中或连接到进程的 rset 所包含的信息。

标志

-f 显示 rset 所有者、组和方式数据。
-v 详细方式。显示 rset、rset 所有者、组和方式数据中包含的资源。
-o 仅显示 rset 所包含的联机资源。缺省显示所有资源。
-p pid 显示连接到进程的有效 rset。
-r rsetname 以系统注册表中的此名称显示 rset。名称由以“/”（斜杠）分隔的名称空间和 rsname 构成。namespace 和 rsname 最多 255 字符。关于 rset 名称的字符集限制的附加信息，请参阅 rs_registername() 服务。
-n namespace 显示系统注册表中 namespace 的所有 rset。
-a 显示系统注册表中所有的 rset。
-P 显示连接到指定进程的分区 rset。

示例

  1. 要显示系统注册表中所有 rset 的所有资源，请输入： 
    lsrset -v -a
  2. 要显示连接到 pid 28026 的有效 rset 的摘要，请输入： 
    lsrset -p 28026
  3. 要显示连接到 pid 28026 的有效 rset 的联机资源，请输入： 
    lsrset -o -p 28026
  4. 要显示连接到 pid 28026 的有效 rset 的所有资源，请输入： 
    lsrset -v -p 28026
  5. 要显示系统注册表中所有的 rsets 的联机资源，请输入： 
    lsrset -a -o

文件

/etc/bin/lsrset 包含 lsrset 命令
