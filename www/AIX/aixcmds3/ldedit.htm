ldedit 命令

用途

修改一个 XCOFF 可执行文件头。

语法

ldedit -b Option [ -V ] File

描述

可以使用 ldedit 命令修改 XCOFF 头中或可执行文件的辅助头中的各种字段。ldedit 命令可以将应用程序标记或取消标记为‘大页数据’程序。ldedit 命令还可以在不重新链接的情况下添加或修改 MAXDATA 和 MAXSTACK 的值。

-b 标志的格式与链接编辑器 ld 命令使用的格式类似。-b 标志可以在命令行上多次使用。

如果未指定标志，则 ldedit 命令使用标准错误输出显示用法消息。

标志

-b Option 修改由 Option 指定的可执行文件。Option 的可能值是：
lpdata
将文件标记为‘大页数据’可执行文件。
nolpdata
将文件取消标记为‘大页数据’可执行文件。
maxdata：value
设置 MAXDATA 值。Value 当以 0 开始时是个八进制数，当以 0x 开始时是个十六进制数，其他情况是十进制数。
maxdata：value/dsa
设置 MAXDATA 值和 DSA 位。Value 当以 0 开始时是八进制数，当以 0x 开始时是十六进制数，其他情况是十进制数。
maxstack：value
设置 MAXSTACK 值。Value 当以 0 开始时是个八进制数，当以 0x 开始时是个十六进制数，其他情况是十进制数。
-V 在标准错误输出上打印 ldedit 命令的版本。
