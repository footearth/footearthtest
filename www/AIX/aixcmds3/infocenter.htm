infocenter 命令

用途

在浏览器窗口中启动 IBM eServer pSeries 信息中心。

语法

infocenter

描述

infocenter 命令启动浏览器窗口中的 IBM eServer pSeries 信息中心。信息中心是一个中心点，从这里可以访问 AIX 和 IBM eServer pSeries 文档、AIX FAQ、Message Center和入门文档。信息中心还可以通过单击 CDE 工具栏的帮助面板上的信息中心图标来启动。

注:
信息中心只能在对因特网具有访问权的 HTML 浏览器的条件下可用。infocenter 命令（和信息中心图标）激活浏览器并将浏览器指向 信息中心的 Web 地址。

信息中心的 Web 地址在 /usr/lpp/bosinst/bos.vendor.profile 文件中。如果该文件中提供了信息中心的 Web 地址，则它为以下格式：

export INFORMATION_CENTER_URL=information center url

如果/usr/lpp/bosinst/bos.vendor.profile 文件中不存在与此类似的行，或者如果该文件不存在，则用户将接收到一条消息，表示由于找不到信息中心而不能启动信息中心。

文件

/usr/sbin/infocenter 在浏览器窗口中启动信息中心。
/usr/lpp/bosinst/bos.vendor.profile 包含供应商概要文件信息，这些信息包括由 infocenter 命令使用的 Web 地址。
