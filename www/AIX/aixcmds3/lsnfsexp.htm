lsnfsexp 命令

用途

显示以网络文件系统（NFS）导出的目录特征。

语法

/usr/sbin/lsnfsexp [ -c | -l ] [ Directory ] [ -f Exports_file ]

描述

lsnfsexp 命令显示已导出 NFS 的目录特征。Directory 参数指定要显示的路径。如果未指定目录，将显示所有以 NFS 导出的目录。

标志

-c 指定输出应使用冒号格式。
-l （小写的 L）指定应以表格式输出。该标志是缺省的。
Directory 指定要显示的目录。如果未指定目录，将显示所有以 NFS 导出的目录。
-f Exports_file 如果路径非 /etc/exports，那么需指定导出文件的全路径名。

示例

  1. 以冒号格式列出当前以 NFS 导出的全部目录，请输入： 

    lsnfsexp -c
  2. 以冒号格式列出当前以 NFS 导出的全部目录并使用指定的路径名，而不是 /etc/exports，请输入： 

    lsnfsexp -c -f /etc/exports.other

文件

/etc/exports 列出服务器可导出的目录。
