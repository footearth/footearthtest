kmodctrl 命令

用途

装入或卸装内核扩展 /usr/lib/drivers/kmobip6。

语法

kmodctrl [ -k kextname ] [ -luq ]

描述

内核扩展 /usr/lib/drivers/kmobip6 包含对“移动式 IPv6”（Mobile IPv6）功能的支持。为了将系统配置为一个移动式 IPv6 主代理或相应节点必须加载这个内核扩展。正常情况下，如果使用系统管理启动了移动式 IPv6，这个命令将由 /etc/rc.mobip6 脚本自动运行。

标志

-k 为移动性内核扩展指定备用路径。
-l 装入移动性内核扩展。
-q 检查移动性内核扩展是否被检查。
-u 卸装移动性内核扩展。

退出状态

0
这个命令成功完成。
>0
有错误发生。

安全性

必须是 root 用户或 system 组成员来执行这个命令。

示例

  1. 以下示例装入 kmobip6 内核扩展： 
    kmodctrl -l
  2. 下列示例卸装 kmobip6 内核扩展。这个将禁用所有系统上的移动式 IPv6 功能： 
    kmodctrl -u
  3. 以下示例查询是否 kmobip6 内核扩展被装入： 
    kmodctrl -q
