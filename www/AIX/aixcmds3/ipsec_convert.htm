ipsec_convert 命令

用途

把 IP 安全性隧道导出文件转换为一种能被 IBM Secure Network Gateway 导入的格式。

语法

ipsec_convert SNG22 | FW31 [-f export_directory]

描述

IP 安全性允许使用 imptun 命令来导入 IBM Secure Network Gateway2.2 和 IBM Firewall3.1 隧道。然而，这些防火墙产品不允许反向能力。ipsec_convert 命令通过将导出的 IP 安全性隧道转换为 IBM 的防火墙隧道来允许这种能力。已转换的文件放置在当前目录下。

标志

SNG22 | FW31 指定结果文件的格式是 IBM Secure Network Gateway 2.2 还是 IBM Firewall 3.1 的格式。
-f 指定存放导出文件的目录。
