mvfilt 命令

用途

移动过滤规则。

语法

mvfilt -v 4|6 -p p_fid -n n_fid

描述

使用 mvfilt 命令来更改过滤规则表中过滤规则的位置。

标志

-v 过滤规则的 IP 版本。值 4 指定 IP V4 且值 6 指定 IP V6。
-p 过滤规则标识。它指定过滤规则表中规则的前一个位置。 对于 IP V4，由于第一个过滤规则不可移动，值 1 是无效的。
-n 过滤规则标识。它指定过滤规则表中规则在移动后的新位置。 对于 IP V4，由于第一个过滤规则是保留的因而无法移动，值 1 是无效的。
