lsvfs 命令

用途

列出 /etc/vfs 文件中的条目。

语法

lsvfs { -a | VfsName }

描述

lsvfs 命令列出 /etc/vfs 文件中的条目。可以显示关于特定的“虚拟文件系统”（VFS）类型或所有已知的 VFS 类型的信息。

标志

-a 列出 /etc/vfs 文件中的所有节，包括缺省节。

参数

VfsName 指定虚拟文件系统名。

示例

  1. 要列出名为 newvfs 的 vfs 条目，请输入： 

    lsvfs newvfs
  2. 要列出所有 vfs 类型，请输入： 

    lsvfs -a

文件

/etc/vfs 包含虚拟文件系统类型的描述。
