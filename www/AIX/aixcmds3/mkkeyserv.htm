mkkeyserv 命令

用途

为 keyserv 守护进程取消 /etc/rc.nfs 文件中项的注释，并使用 startsrc 命令激活守护进程。

语法

/usr/sbin/mkkeyserv [ -I | -B | -N ]

描述

mkkeyserv 命令取消 /etc/rc.nfs 文件中对 keyserv 守护进程项的注释。mkkeyserv 命令通过使用 startsrc 命令启动守护进程。

您可以使用基于 Web 的系统管理器（wsm）中的文件系统应用程序来更改文件系统特性。您也可以使用“系统管理接口工具”（SMIT）smit mkkeyserv 快速路径来运行此命令。

标志

-I 取消 /etc/rc.nfs 文件中项的注释以在下一次系统重启时启动 keyserv 守护进程。
-B 取消 /etc/rc.nfs 文件中项的注释以启动 keyserv 守护进程，并使用 startsrc 命令启动 keyserv 守护进程。这个标志是缺省值。
-N 使用 startsrc 命令启动 keyserv 守护进程。此标志不更改 /etc/rc.nfs 文件。

示例

要修改 /etc/rc.nfs 文件以在下一次系统重新启动时激活 keyserv 守护进程，请输入：

mkkeyserv -I

文件

/etc/rc.nfs 包含 NFS 和 NIS 守护进程的启动脚本。
