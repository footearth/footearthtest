lsvmode 命令

用途

显示 X 服务器上的当前视频方式。

注：这个命令只有在 X 服务器正在运行时才可用。

语法

lsvmode

描述

lsvmode 命令显示 X 服务器使用的当前输出设备和视区大小。

安全性

访问控制：任何用户

审计事件：无

退出状态

返回以下退出值：

0 成功完成。
>0 发生错误。

示例

显示 X 服务器上的当前视频方式。

lsvmode

显示内容大致如下：

Current video mode information
Logical screen size   [1024x768]
Viewport size        [ 640x480]
Vertical sync. (Hz)       [60]
Active output device  [LCD][CRT]

文件

/usr/bin/X11/lsvmode 包含 lsvnode 命令。
