inutoc 命令

用途

给包含备份格式文件安装映像的目录创建一个 .toc 文件。这个命令由 installp 命令和安装脚本来使用。

语法

inutoc [ Directory ]

描述

inutoc 命令在 Directory 下创建 .toc 文件。如果 .toc 文件已经存在，它将带着新信息重新创建。缺省的安装映像Directory 是 /usr/sys/inst.images。inutoc 命令给 Directory 下的每个安装映像添加 .toc 文件中内容项表。

installp 命令和 bffcreate 命令在创建和使用目录中的安装映像而不带 .toc 文件时自动调用这个命令。

错误代码

INUBADIR 用法错误或者 Directory 没有指定一个目录。
INUCHDIR 无法将目录更改为 Directory。
INUCRTOC 无法创建 .toc 文件。
INUGOOD 没有出现错误。
INUSYSFL 系统调用失败。

安全性

权限控制：只有 root 用户才能运行该命令。

示例

  1. 要给 /usr/sys/inst.images 目录创建 .toc 文件，请输入： 

    inutoc
  2. 要给 /tmp/images 目录创建一个 .toc 文件，请输入： 

    inutoc /tmp/images

文件

/usr/sys/inst.images 创建 .toc 文件的缺省目录。
.toc 在指定目录下用该命令创建的文件。
