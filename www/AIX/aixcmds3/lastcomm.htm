lastcomm 命令

用途

显示关于最后执行的命令的信息。

语法

lastcomm [ Command ] [ Name ] [ Terminal ]

描述

lastcomm 命令以逆向时间顺序显示关于仍记录在 /var/adm/pacct 目录下的摘要文件中的先前执行的命令的信息。在能运行 lastcomm 命令前，需要运行 /usr/sbin/acct/startup 命令。

lastcomm 命令显示的列表被限制在以下范围：

  • 由 command 参数指定的命令。
  • 由 Name 参数指定的用户执行的命令。
  • 从由 Terminal 参数指定的终端发出的命令。

    终端可被全命名或简写为 tty。例如，可指定 tty0 终端或 0 终端。

对每个进程，显示下列信息：

  • 运行进程的用户的名字。
  • 当命令执行时记帐设施收集的任何标志。以下是有效的标志：
    S 根用户执行这个命令。
    F 在创建子进程后这个命令执行，但没有跟着的子例程。
    C 这个命令在 PDP-11 兼容模式下运行。
    D 这个命令终止并生成一个核心文件。
    X 这个命令带着一个信号终止。
  • 这个进程被调用下的命令的名字。
  • 这个进程用的 CPU 时间。
  • 进程开始的时间。

示例

  1. 显示所有记录在/var/adm/pacct 文件中的先前执行命令，输入： 

    lastcomm
  2. 要显示 ttyd0 终端上的 root 用户执行的名为 a.out 的命令的信息，输入： 

    lastcomm a.out root ttyd0

文件

/usr/bin/lastcomm 包含 lastcomm 命令。
/var/adm/pacct 包含当前记帐摘要文件的目录。
