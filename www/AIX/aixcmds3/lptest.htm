lptest 命令

用途

生成行式打印机波纹模式

语法

lptest [ ;Length Count ]

描述

lptest 命令在像终端或者打印机这样的标准输出设备上写传统的“ripple”测试模式。在 96 行之内，该模式将在每个位置打印所有的 96 个可打印 ASCII 字符。尽管初始创建是用来测试打印机的，但是波纹（ripple）模式非常适用于测试终端、驱动调试用途的终端端口或者任何其它的需要快速提供随机数据的任务。

使用 lptest 命令，如果缺省的长度 79 不合适的话可以指定输出行长度。如果缺省的 Count 参数 200 不合适，还可以指定生成的输出行的数目。注意，如果指定了 Count 参数，也必须指定 Length。

示例

要显示或打印第 80 列、 100 行测试输出到标准输出，请输入：

lptest 80 100
