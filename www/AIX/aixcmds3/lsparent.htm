lsparent 命令

用途

显示可能接受指定连接类型或设备的父设备。

语法

lsparent { -C | -P } { -k ChildConnectionKey | -l ChildName } [ -f File ] [ -FFormat] [ -h ] [ -H]

描述

lsparent 命令列出设备配置数据库中的设备，设备配置数据库可以接受给定的设备作为子设备，由 -l ChildName 标志指定，或接受由 -k ChildConnectionKey 标志指定的子设备连接的给定类型。

可以用两种方法之一显示缺省输出。可以使用 -C 标志，显示定制设备对象类的设备的缺省输出信息，诸如名称、状态、位置和描述，或使用 -P 标志，显示预定义设备对象类的设备的缺省输出信息，诸如类、类型、子类和描述。要覆盖这两种缺省输出，可以使用 -F Format 标志来显示用户格式的字符串指定的输出。Format 参数是列名引用列表，这些列表以非字母数字字符分隔，也可能由它们结尾。

在命令行中或者从指定的 File 参数可以提供该标志。

标志

-C 列出在定制设备对象类中的设备信息。显示的信息可能来自定制的和预定义的设备对象类。该标志不能和 -P 标志一起使用。
-f File 从 File 变量中读取所需的标志。
-F Format 以用户指定格式显示输出，其中 Format 变量是来自预定义设备对象类或定制设备对象类的列名引用列表，这些列表以非字母数字字符分隔，也可能由它们结尾。使用空格作为分隔符，lsparent 命令用对齐的列显示输出。除两个对象类中的栏名外，特殊的名称 description 可用来显示设备的文本描述。
-H 显示列输出上方的头部分。
-h 显示命令用法消息。
-k ChildConnectionKey 指定标识子设备的设备子类名称的连接键。该标志不可和 -l 标志一起使用。
-l ChildName 指定可能子设备的逻辑名称。该标志不可和 -k 标志一起使用。
-P 列出在预定义设备对象类中的设备信息。显示的信息来源于定制的或预定义的设备对象类。 该标志不可和 -C 标志一起使用。

示例

  1. 要列出在接受 RS-232 设备的定制设备对象类中的可能的父设备，请输入： 
    lsparent -C -k rs232
    系统显示与以下类似的消息： 
    sa0 可用 01-S1 标准 I/O 串口
sa1 可用 01-S2 标准 I/O 串口
sa2 可用 10-68 IBM 8 端口 EIA-232/RS-422A（PCI）适配器
sa3 可用 10-70 IBM 8 端口 EIA-232/RS-422A（PCI）适配器
sa4 可用 10-78 IBM 8 端口 EIA-232/RS-422A（PCI）适配器
sa5 可用 20-58 IBM 8 端口 EIA-232/RS-422A（PCI）适配器
  2. 要列出接受 RS-232 设备的预定义设备对象类中的父设备的可能类型，请输入： 
    lsparent -P -k rs232
    系统显示与以下类似的消息： 
    适配器       pnp501   isa_sio  标准 I/O 串行端口
adapter        4f111100       pci        IBM 8 端口 EIA-232/RS-422A（PCI）适配器
集中器       16c232   sync_pci 用于 128 端口适配器的 16 端口 RAN EIA-232
集中器       16e232   sync_pci 用于 128 端口适配器的 16 端口增强 RAN EIA-232
  3. 要列出接受磁带设备 rmt0 作为子设备的定制设备对象类中的可能的父设备，请输入： 
    lsparent -C -l rmt0
    系统显示与以下类似的消息： 
    scsi2 可用      20-60 Wide/Ultra-2 SCSI I/O 控制器
scsi3 可用      20-61 Wide/Ultra-2 SCSI I/O 控制器
scsi1 可用      10-88 Wide/Ultra-2 SCSI I/O 控制器
scsi0 可用      10-60 Wide/Fast-20 SCSI I/O 控制器
  4. 要列出接受磁带设备 rmt0 作为子设备的预定义设备对象类中的父设备的可能类型，请输入： 
    lsparent -P -l rmt0
    系统显示与以下类似的消息： 
    适配器  sym896 pci Wide/Ultra-2 SCSI I/O 控制器
适配器  sym895 pci Wide/Ultra-2 SCSI I/O 控制器
适配器  sym875 pci Wide/Fast-20 SCSI I/O 控制器

文件

/usr/sbin/lsparent 包含 lsparent 命令。
