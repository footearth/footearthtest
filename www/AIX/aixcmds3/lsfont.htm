lsfont 命令

用途

将可用字体列出到显示器上。

语法

lsfont [ -l]

描述

lsfont 命令将可用字体列表显示在显示器上。字体标识符可以帮助用户使用 chfont 命令修改字体。

可以使用基于 Web 的系统管理器设备应用程序（wsm devices 快速路径）来运行此命令。也可以使用系统管理接口工具（SMIT）smit lsfont 快速路径来运行此命令。

标志

-l 指定从数据格式中移去所有头信息。

示例

要将所有可用字体列到显示器上显示，请输入：

lsfont

下例显示了每个可用字体的字体标识符、字体名、字形大小和字体编码：

字体  文件           图象字符  字体
标识  名             大小      编码
====  ==============  =====  =========
0     Erg22.iso1.snf  12x30  ISO8859-1
1     Erg11.iso1.snf   8x15  ISO8859-1

有关可用字体的详细信息，请参阅 AIX 系统中的文本字体。

文件

/bin/lsfont 包含 lsfont 命令。
/usr/lpp/fonts 包含字体目录。

相关信息

chfont 命令、mkfont 命令。
