lsquedev 命令

用途

显示设备节名称。

语法

lsquedev [ -c ] -qName -d Name

描述

lsquedev 命令显示队列节名称和 /etc/qconfig 文件的相关属性。

标志

-c 指定 SMIT 使用的冒号输出格式。
-dName 指定要显示的设备节的 Name 变量。
-q Name 指定包含要显示的设备节的队列 Name 变量。

示例

  1. 要显示在 lp0 队列中的设备节 dlp0，请输入：
    lsquedev  -q lp0  -d dlp0
    显示类似如下列表： 

    dlp0:
FILE = /dev/lp0
BACKEND = /usr/lib/lpd/piobe
  2. 要以冒号格式显示在 lp0 队列中的设备节 dlp0，请输入：
    lsquedev  -c  -qlp0  -d dlp0
    显示类似如下列表： 

    file:access:feed:header:trailer:backend:align
dlp0:/dev/lp0:read:never:never:never:/usr/lib/lpd/piobe:TRUE

文件

/usr/bin/lsquedev 包含 lsquedev 命令。
/etc/qconfig 包含配置文件。
