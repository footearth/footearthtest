learn 命令

用途

提供使用文件、编辑器、宏和其他功能部件的计算机辅助指令。

语法

learn[- Directory] [ Subject[ LessonNumber]]

描述

learn 命令提供使用文件、编辑器、宏和其它功能部件的计算机辅助指令。第一次调用这个命令时，系统提供关于 learn 命令的介绍性信息。否则，learn 命令在您离开最后一个 learn 命令对话的地方开始。

能通过指定 Subject 参数绕过 learn 命令的缺省操作。learn 命令和指定的主题的第一课同时启动。可指定任何以下主题：

  • 文件
  • 编辑器
  • 更多文件
  • 宏
  • EQN（查询字符）
  • C（语言）

    注： 仅可在一个能有 1/2 线运动的硬拷贝终端上运行 EQN 课程。/usr/share/lib/learn/eqn/Init 文件包含一个详细的支持终端的列表。

当输入 learn 命令，系统搜索 /usr/share/lib/learn 目录查找适当的课程文件。用 -Directory 标志来标识不同的搜索目录。

子命令

  • bye 子命令终止 learn 命令会话。
  • where 子命令告诉您您的进程； where m 子命令提供更详细信息。
  • again 子命令重新显示课程的文本。
  • again LessonNumber 子命令让您回顾课程。
  • hint 子命令打印用来评估响应的课程脚本的的最后部分；hint m 子命令打印整个课程脚本。

参数

-Directory 允许你指定一个不同的搜索目录。缺省情况下，系统在 /usr/share/lib/learn 目录下搜索课程文件。
LessonNumber 标识课程的编号。
Subject 指定您想要指示信息的主题。

示例

取得关于文件的在线课程，输入：

learn files

系统启动 learn 程序并为怎样使用程序显示指示信息。

文件

/usr/share/lib/learn 包含所有相关目录和文件的文件树。
/tmp/pl* 包含实际目录。
$HOME/.learnrc 包含启动信息。
