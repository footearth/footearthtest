mhl 命令

用途

产生消息的格式化列表。

语法

mhl [ -form FormFile ] [ -folder +Folder ] [ -moreproc Command | -nomoreproc [ -bell | -nobell ] [ -clear | -noclear ] ] [ -length Number ] [ -width Number ]

描述

mhl 命令创建消息的格式化列表。 这个命令通常是通过 showproc: 概要文件项或其它 MH 命令中的 -showproc 标志来启动。当显示消息时，mhl 命令使用格式文件中所列的指令。如果指定多于一个消息，那么 mhl 命令在每屏消息显示前提供一个提示符。

如果指定 -nomoreproc 标志，那么 mhl 命令会提示用户按下“返回”（也可以是 Ctrl-D 按键顺序）来查看下一条消息。要停止当前消息输出并接收下一条消息的提示符，请按下 Ctrl-D 按键顺序。按下“退出”按键顺序以停止命令输出。

注：要使用 mhl 命令，必须建立想要处理当前目录的文件夹。

标志

-bell 在每页结尾产生一个报警符。当指定了 -nomoreproc 标志或定义了 moreproc: 概要文件项但它们为空时，-bell 标志为缺省值。
-clear 当输出设备是显示器时，每显示一页后清除屏幕。mhl 命令使用 $TERM 环境变量来判断显示器的类型。当输出设备不是显示器时，-clear 标志在每个消息的结尾插入一个换页符。这个标志仅在 moreproc: 概要文件项已定义且为空时影响 mhl 命令。
-folder +Folder 识别用于 mhl.format 文件的 MessageName: 项的文件夹。缺省值为 $mhfolder 环境变量的值。
-form FormFile 指定包含备用输出格式的文件。缺省格式在 UserMHDirectory/mhl.format 文件中描述。如果此文件不存在，则 mhl 命令使用 /etc/mh/mhl.format 文件中描述的系统缺省格式。
-help 列出命令语法、可用的开关（切换）和版本信息。

注：对于 MH，此标志的名称必须全部拼写。
-length Number 设置输出屏幕的长度。缺省值为 $TERM 环境变量表示的值。如果此值不适当，则缺省值为 40 行。
-moreproc Command 使用 Command 变量的值代替 $HOME/.mh_profile 文件中指定的 moreproc: 项的值。
-nobell 在每页的结尾禁止产生报警符。只有当输出设备是显示器，使用了 -nomoreproc 标志，或 moreproc: 概要文件项已定义且为空时，此标志才影响 mhl 命令。
-noclear 当输出设备是显示器时，在每页的结尾不清除屏幕。当输出设备不是显示器时，-clear 标志不在每个消息结尾插入格式反馈符。当使用了 -moreproc 标志或 moreproc: 项已定义且为空时，此标志为缺省值。
-nomoreproc 设置 moreproc: 项为空值。
-width Number 设置输出屏幕的宽度。缺省值为 $TERM 环境变量表示的值。如果此值不适当，则缺省值为 80 行。

概要文件项

在 UserMHDirectory/.mh_profile 文件中可以查找到以下条目：

moreproc: 指定与用户进行通信的交互式程序。

示例

  1. 要列出 inbox 文件夹中的消息 5，请将目录更改到 inbox： 

    cd /home/mickey/Mail/inbox

    然后输入：

    /usr/lib/mh/mhl 5

    显示与以下内容相似的信息：

    --- Using template MHL.FORMAT ---
Date:
   
To:
cc:
   
From:
Subject:
   
消息文本
  2. 要显示多于一个的消息，请输入： 

    /usr/lib/mh/mhl 5 6 7

文件

$HOME/.mh_profile 包含 MH 用户概要文件。
/etc/mh/mhl.format 定义缺省的 MH 消息模板。
UserMHDirectory/mhl.format 指定用户的缺省消息模板。（如果它存在，则覆盖缺省的 MH 消息模板。）
/usr/lib/mh/mhl 包含 mhl 命令。
