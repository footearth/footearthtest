swapoff 命令

用途

取消激活一个或多个调页空间。

语法

swapoff DeviceName { DeviceName ...}

描述

swapoff 命令取消激活一个或多个调页空间。 DeviceName 指定调页空间。要取消激活：

  • 调页空间必须是先前通过 swapon 命令激活的。
  • 在剩余的调页空间里必须具有足够的空间。

退出状态

值 描述
0 取消激活成功，调页状态设置为“不活动”状态。
1 会显示以下消息： 

swapoff：不能取消激活调页空间 DeviceName
2 在剩余的调页空间里没有足够的空间，取消激活没有完成，会显示以下消息： 

“swapoff：不能取消激活调页空间 DeviceName：文件系统没有足够的空间。”
3 调页空间的用户页面出现 I/O 错误，会显示以下消息： 

swapoff：调页空间 DeviceName 取消激活暂挂：在用户支持页面遇到 I/O 错误。

建议操作：

  • 检查错误日志。
  • 使用 chps 命令取消激活调页空间以便在下一步重新引导。
  • 重新引导系统。
4 调页空间的系统页面发生 I/O 错误，会显示以下消息： 

swapoff：取消激活调页空间 DeviceName 暂挂：系统支持页面遇到 I/O 错误。系统可能崩溃。

建议操作：

  • 检查错误日志。
  • 使用 chps 命令取消激活调页空间以便在下一步重新引导。
  • 重新引导系统。
