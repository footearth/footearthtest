sccsdiff 命令

用途

比较 SCCS 文件的两个版本。

语法

sccsdiff -rSID1 -rSID2 [  -p ] [  -sNumber ] File ...

描述

sccsdiff 命令读取“源代码控制系统”（SCCS）文件的两个版本，对它们进行比较，并将两个版本的差别写到标准输出中。可以指定任意 SCCS 文件数，但是所有的文件要使用相同的参数。

标志

-p 通过 pr 命令进行输出。
-rSID1 将 SID1 指定为 SCCS 文件的一个 delta，以便用 sccsdiff 命令进行比较。
-rSID2 将 SID2 指定为一个 SCCS 文件的另一个 delta，以便用 sccsdiff 命令进行比较。
-sNumber 指定文件节大小，以便用 bdiff 命令传递给 diff 命令。这对于 diff 命令由于高系统负载而失效尤为有用。

示例

要显示 SCCS 文件 s.test.c V1.1 和 V1.2 的差别, 请输入：

sccsdiff -r1.1 -r1.2 s.test.c

文件

/usr/bin/sccsdiff 包含 SCCS sccsdiff 命令。sccsdiff 命令支持多字节字符集（MBCS）数据作为文件名。
