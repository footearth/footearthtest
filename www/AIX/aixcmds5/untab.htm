untab 命令

用途

将制表符改为空格

语法

untab [ FileName ... ]

描述

untab 命令读取由 FileName 参数或标准输入指定的文件，并用空格字符替换输入中的制表符。如果用 FileName 参数指定文件，untab 命令将结果文件写回成原文件。如果输入为标准输入，untab 命令写成标准输出。untab 命令假定制表符停止位设置在每隔八列处，并从第九列开始。为 FileName 参数指定的文件名长度不能超过 PATH_MAX-9 个字节。

示例

要用空格字符替换 File 文件中的制表符，输入：

untab File

文件

/usr/bin/untab 包含 untab 命令。
