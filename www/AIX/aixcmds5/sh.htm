sh 命令

用途

调用缺省 shell。

语法

请参阅 ksh 命令的语法。将 /usr/bin/sh 文件链接到 Korn shell。

描述

sh 命令调用缺省 shell 并使用它的语法和标志。链接到 /usr/bin/sh 路径的 shell 是缺省的 shell。操作系统的标准配置把 /usr/bin/sh 路径链接到 Korn shell 。关于 Korn shell 功能的具体信息，请参阅《AIX 5L V5.2 系统用户指南：操作系统与设备》中的『Korn Shell 或 POSIX Shell 命令』。

标志

请参阅 Korn shell（ksh 命令）的标志。

文件

/usr/bin/sh 包含 sh 命令。

相关信息
