style 命令

用途

分析文档的表面特征。

语法

style [  -a ] [  -e ] [  -lNumber ] [  -ml ] [  -mm ] [  -p ] [  -P ] [  -rNumber ] File ...

描述

style 命令分析英语文档的写入样式的表面特征。它会报告可读性、语句长度和结构、单词长度和用法、动词类型以及语句开头。因为 style 命令在查看文本前先运行 deroff 命令，则应该将包含相应的格式化信息的头文件作为输入的一部分包括。

注：使用非标准格式化的宏可能会导致不正确的语句中断。

标志

-a 打印所有语句以及它们的长度和可读性索引。
-e 打印所有以虚词（比如“There are”）开头的语句。
-lNumber 打印所有多于 Number 参数所指定的数字的语句。
-ml 使 deroff 命令跳过列表；如果文档包含许多语句片断的列表，则使用 -ml。
-mm 覆盖缺省的 ms 宏数据包。
-p 打印所有包含被动动词的语句。
-P 打印文档中的单词的语音部分。
-rNumber 打印所有可读性索引大于 Number 的语句。
