9.6 gawk Can Speak Your Language

As of version 3.1, gawk itself has been internationalized using the GNU gettext package. (GNU gettext is described in complete detail in GNU gettext tools.) As of this writing, the latest version of GNU gettext is version 0.11.5.

If a translation of gawk's messages exists, then gawk produces usage messages, warnings, and fatal errors in the local language.
